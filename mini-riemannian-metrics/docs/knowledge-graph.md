# Knowledge Graph — Mini Riemannian Metrics

## L1: Core Definitions
- `MetricTensor n` — Riemannian metric (symmetric, positive definite (0,2)-tensor)
- `RiemannianManifold n` — manifold with Riemannian metric
- `flatMap` / `sharpMap` — musical isomorphisms
- `inverseMetric` — inverse metric tensor
- `curveLength` / `curveEnergy` — length and energy functionals
- `riemannianDistance` — induced distance function
- `volumeElement` / `totalVolume` — Riemannian volume
- `gradient` / `divergence` / `laplaceBeltrami` — metric differential operators
- `christoffelFirstKind` / `christoffelSecondKind` — connection coefficients
- `exponentialMap` — geodesic exponential
- `JacobiField` — Jacobi field along geodesic

## L6: Canonical Examples
- Euclidean space R^n (flat metric, K=0)
- Round sphere S^n (constant K=1/r^2)
- Hyperbolic space H^n (constant K=-1)
- Flat torus T^n = R^n / Z^n
- Real projective space RP^n = S^n / {+/-1}
- Complex projective space CP^n (Fubini-Study, Kahler-Einstein)
- Schwarzschild metric (spherically symmetric vacuum)
- Zoll surfaces (all geodesics closed)
- Exotic spheres (Milnor's 28)
- Berger spheres (collapsing)
- Gromoll-Meyer exotic sphere (K>=0)
- Isospectral manifolds (Sunada, Milnor 16-tori)
