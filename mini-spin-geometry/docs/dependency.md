## Knowledge Dependencies

Spin geometry requires understanding of:
1. Linear algebra (vector spaces, quadratic forms)
2. Abstract algebra (associative algebras, representations)
3. Differential geometry (manifolds, connections, curvature)
4. Algebraic topology (characteristic classes, cobordism)
5. Functional analysis (elliptic operators, index theory)
